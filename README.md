# Portfolio Admin Demo

A simple frontend-only admin panel for editing portfolio content on admin.mahialamazad.com.

## What it shows

- Profile editing
- Skills editing
- Education editing
- Live preview of the public profile layout
- SPA sub-routes for `/profile`, `/skills`, and `/education`

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Notes

- This is a frontend demo only.
- The save and publish buttons are UI placeholders for portfolio presentation.
- Visiting `/` redirects to the last used section or `/profile`.
